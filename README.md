#PostGet
##Generic Post Get Orchestrator based on Clexical

###Building
To compile and build a runnable version:
```
./rebar get-deps
./rebar compile
cd rel
../rebar generate
```

###Running

###Start
```
./postget start
>
```
###Stop
```
./postget stop
>
```

###Probing
```
./postget ping
pong
>
```

###Logging
Info Logs at
```
./log/info.log
```

###Debug Logs at
```
./log/debug.log
```
